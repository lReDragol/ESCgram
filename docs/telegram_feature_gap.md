# Telegram Gap Audit

## Что уже есть в ESCgram

- Базовый список чатов, история сообщений, медиа, голосовые, кружки.
- Ответы, пересылка, выделение нескольких сообщений, удаление.
- Реакции на сообщения.
- Автоскачивание медиа и локальный кэш своих файлов.
- AI-режим по чатам с отдельной памятью и межчатовым контекстом.

## Что добавлено в этой доработке

- Шапка активного чата с аватаром, названием, метаданными и меню действий.
- Профиль чата/канала/пользователя по клику на шапку.
- Статистика по чату и отдельному сообщению из интерфейса.
- Хранение реакций, опросов, просмотров и пересылок в локальной БД.
- Журнал удалённых сообщений для чатов, где удаление пришло через Telegram updates.
- Подкачка недавних эмодзи пользователя из Telegram-истории в панель выбора.
- Батч-подкачка reply-preview и уменьшение синхронных запросов к storage при открытии истории.

## Чего всё ещё нет относительно официального Telegram

### Коммуникация

- Папки/фильтры диалогов уровня Telegram.
- Темы и топики в группах.
- Закрепление нескольких сообщений с навигацией по ним.
- Отправка опросов, викторин, геопозиции, контактов.
- Редактор медиа перед отправкой уровня Telegram.
- Отправка альбомов с полной раскладкой превью и reorder до отправки.

### Сообщения

- Полноценный рендер premium/custom emoji как медиа-эмодзи Telegram.
- Inline-клавиатуры/кнопки ботов в полном объёме.
- Живое обновление счётчиков реакций/опросов без повторной загрузки истории.
- Цепочки обсуждений и комментариев у каналов.
- Планировщик сообщений и отправка «когда онлайн».

### Чаты и профили

- Полноценные страницы профиля пользователя/группы/канала со всеми разделами.
- Управление участниками, правами, банами, медиа общим списком.
- Общие медиа/файлы/ссылки/музыка как отдельные вкладки профиля.
- Поиск по участникам и сообщениям уровня Telegram Desktop.

### Статистика и модерация

- Полная серверная статистика Telegram для каналов/групп с историческими графиками.
- Выявление накруток по user-level данным реакций и голосований.
- Просмотр списка проголосовавших/реагировавших там, где Telegram API это позволяет.
- Системный аудит удалений/правок с длинной историей и экспортом.

### UX и платформа

- Виртуализация длинной истории на уровне списка виджетов.
- Полноценная адаптация под Linux desktop-specific интеграции.
- Горячие клавиши, drag/drop и контекстные действия на уровне официального клиента.

## Приоритет на следующие этапы

1. Виртуализация списка сообщений и дальнейшее снижение времени открытия чата.
2. Live-обновление реакций/опросов/правок без переоткрытия.
3. Полные страницы профиля чата с вкладками медиа/файлы/ссылки.
4. Рендер и отправка Telegram custom emoji / sticker emoji.
5. Продвинутая аналитика групп и каналов с отдельной БД сигналов по подозрительной активности.
